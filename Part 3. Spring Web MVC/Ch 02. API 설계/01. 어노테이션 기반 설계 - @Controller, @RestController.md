# 어노테이션 기반 설계
## 컨트롤러 클래스
- MVC 패턴 중 핸들러 메소드를 포함하는 컨트롤러 빈을 만드는 과정
  - @Controller
  - @RestController = @ResponseBody + @Controller

## 핸들러 메소드(Handler Method)
- 스프링 웹 서비스가 받는 URL 요청을 컨트롤러 클래스의 특정 메소드에 매핑하는 과정
  - @RequestMapping
  - @GetMapping
  - @PostMapping
  - @PutMapping
  - @DeleteMapping
  - @PatchMapping
